# @udos/viewport-renderer

Renderer package for grid data from @udos/gridcore.

## Exports

- ViewportWidget (abstract)
- TerminalWidget (DOMViewport)
- TeletextWidget (CanvasViewport)
- CanvasViewport and DOMViewport
- PETME64/Teletext50 font loaders
- USX/C64/Teletext/NES palettes
- CRT effects and animation helpers

## Build

```bash
npm run build
```
