# @udos/gridcore

Pure grid algebra package for uDos. This package contains coordinate math, viewport calculations, and teletext geometry primitives. It has no rendering dependencies.

## Highlights

- Cell -> Grid -> Layer -> Cube data model
- uCode <-> lat/lon conversion helpers
- Viewport calculations for 80x24, 60x24, custom modes
- Teletext geometry helpers (2x3 mosaic)
- QR slot placement helpers (3x3)

## Install

```bash
npm install @udos/gridcore
```

## Build

```bash
npm run build
```
