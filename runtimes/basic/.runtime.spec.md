# .runtime Nugget Format

A `.runtime` nugget is a self-describing unit of BBC BASIC program
source that can be executed by the uCode BASIC runtime.  It is a
gzipped tarball with the following layout:

```
example.runtime
  ├── meta.json          # nugget manifest
  ├── program.bas        # BBC BASIC source
  ├── assets/            # optional embedded assets
  │   ├── sprites/       # .png or .gif sprite sheets
  │   └── audio/         # .wav or .ogg samples
  └── snapshot.json      # optional variable snapshot
```

## meta.json

```json
{
  "format": "runtime-nugget/1.0",
  "name": "Example Program",
  "author": "uCode",
  "created": "2026-07-02T00:00:00Z",
  "engine": "bbcsdl-1.36a",
  "viewport": {
    "cols": 80,
    "rows": 24,
    "cellSize": 24
  }
}
```

## program.bas

Standard BBC BASIC source, optionally tagged with line numbers.  The
runtime feeds this directly into the BBCSDL engine via the bridge.

## snapshot.json

Optional variable register state matching the `VariableSnapshot`
format defined in `bridge/variables.py`.  The runtime can restore
this state before executing the program for session-resume support.
