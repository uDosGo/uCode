# uPlace — Cloud Platform

**Next extension from PublishLane. Cloud services for the uDos ecosystem.**

## Scope

- **udo.place** — Cloud platform implementation
- **udo.guide** — Community platform
- **User Management** — Authentication and authorization
- **Cloud Grid** — Global distribution network
- WordPress/headless CMS
- User identity (user-id)
- Global grid infrastructure
- AWS and Google Cloud services

## Architecture

```
uPlace/
├── identity/       # User management & auth
├── grid/           # Global distribution network
├── cms/            # WordPress/headless
├── cloud/          # AWS & GCP service wrappers
└── docs/           # Platform documentation
```
