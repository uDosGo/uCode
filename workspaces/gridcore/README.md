# GridCore Workspace

Live world-building asset root for GridSmith.

## Structure

- `worlds/` world manifests grouped by domain
- `grids/` templates, imports, and exports
- `layers/` layer-specific artifacts
- `fonts/` imported, generated, and sprite font assets
- `maps/` region, city, and dungeon maps
- `scripts/` BASIC, AMOS, and generated scripts

This workspace is intended to stay inside the live uCode monorepo so world assets, runtimes, and packages evolve together.
